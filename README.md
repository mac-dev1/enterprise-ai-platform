# 🤖 Enterprise AI Assistant

An AI-powered internal assistant using Retrieval-Augmented Generation (RAG) to answer company-related questions.

## 🚀 Features

- RAG pipeline with document retrieval
- PDF processing and chunking
- Conversational memory with summarization
- FastAPI backend
- Custom chat frontend
- Local LLM using Ollama

## 🧠 Architecture

- FastAPI (API layer)
- RAG (retrieval + context building)
- Ollama (LLM inference)
- Sentence Transformers (embeddings)

## 📦 Setup

```bash
git clone <repo>
cd enterprise-ai-platform
pip install -r requirements.txt
```

## Run
```bash
python run.py
